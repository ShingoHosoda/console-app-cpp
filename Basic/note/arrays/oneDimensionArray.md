# 基本

## $1$次元配列配列の操作に関して

## 課題

$1$次元配列の操作方法の知識不足

## 目的

アプリケーション開発を通じて$1$次元配列の操作方法を理解する。

### 数値の操作

#### 入力要件1

入力はメニュー選択形式

|     入力名     |         入力詳細         |    コマンド    |
| :------------: | :----------------------: | :------------: |
|      追加      |       追加する数値       |   ```add```    |
|      削除      |       削除する数値       |  ```remove```  |
|      更新      | 更新前の数値、更新後の値 |  ```update```  |
|      検索      |       検索する数値       |   ```find```   |
| ソート（昇順） |           なし           | ```sortAsc```  |
| ソート（降順） |           なし           | ```sortDesc``` |
|      表示      |           なし           |  ```print```   |
|      終了      |   変更があったかどうか   |   ```exit```   |

#### 出力要件1

|     出力名     |         出力詳細          |    戻り値     |
| :------------: | :-----------------------: | :-----------: |
|      追加      | trueもしくはfalseを出力。 | ```boolean``` |
|      削除      | trueもしくはfalseを出力。 | ```boolean``` |
|      更新      | trueもしくはfalseを出力。 | ```boolean``` |
|      検索      | trueもしくはfalseを出力。 | ```boolean``` |
| ソート（昇順） |           なし            |  ```void```   |
| ソート（降順） |           なし            |  ```void```   |
|      表示      |           なし            |  ```void```   |
|      終了      | trueもしくはfalseを出力。 |  ```boolean```   |
